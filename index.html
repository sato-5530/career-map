経験はまだ浅いですが、異業種での業務を通して培った「相手の意図を正確にくみ取り、適切に伝える力」を活かし、チームの中でも円滑な連携を図ることができています。

感性を活かして集中力を保ちながら業務に取り組む姿勢は、今後さらに磨いていきたいと考えています。

将来的には、自分自身が発信源となり、「自分にしかできない仕事」に取り組めるエンジニアを目指しています。

📊業務経歴

【プロジェクト概要】

ボーイング777の製造業務における品質評価業務に従事。最終的な組立前検査や、不合格品が紛れ込んでいないかの確認を通じて、航空機部品の高精度な品質保証を担当。

【担当業務】

  • ✓ 役割：メンバー
  • ✓ 工程：評価、検査、品質確認、帳票作成

【業務詳細】

  • ✓ 三次元測定機を用いた部品の精密測定・評価
  • ✓ 組立直前の製品に対する最終検査業務
  • ✓ ミリ単位以下のわずかなズレの発見と記録
  • ✓ 不合格品の選別および救済措置に関する書類作成
  • ✓ 品質に関する各種帳票・報告書作成
  • ✓ 出荷前検査で不要部品の付着を発見
  • ✓ 美術で培った観察眼を活かした品質確保

【実績】

  • 🏆 出荷間際に不要部品の付着を発見し、重大な品質トラブルを未然に防止
  • 🏆 見やすい帳票作成により、現場でのレスポンス向上を実現
  • 🏆 観察力を活かし、チーム内でも高精度な評価業務に貢献

【使用ツール】

  • 🔧 三次元測定機
  • 🔧 Excel
  • 🔧 PowerPoint
